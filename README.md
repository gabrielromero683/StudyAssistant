# Study Assistant

## Descripción

Study Assistant es una aplicación de consola desarrollada en Python para ayudar a organizar notas y mostrar frases motivadoras.

## Funciones

- Agregar una nota.
- Ver todas las notas guardadas.
- Mostrar una frase motivadora.
- Salir del programa.

## Cómo ejecutar

1. Tener Python instalado.
2. Ejecutar:

```bash
python main.py
```
