# StudyAssistant

StudyAssistant es un proyecto desarrollado en Python con el propósito de brindar apoyo al estudio mediante herramientas sencillas y fáciles de usar. Además de ayudar a organizar información y realizar actividades relacionadas con el aprendizaje, el proyecto incluye un pequeño apartado de concienciación sobre la protección del medio ambiente, promoviendo hábitos responsables y sostenibles.

## Funciones

- Apoyo al estudio mediante herramientas básicas.
- Organización de información y notas.
- Interfaz sencilla y fácil de utilizar.
- Consejos sobre protección ambiental para fomentar el cuidado del planeta.

## Archivos del proyecto

- **main.py**: archivo principal del programa.
- **requirements.txt**: contiene las dependencias necesarias para ejecutar el proyecto.
- **notes.txt**: almacena notas o información utilizada por el programa.
- **README.md**: documentación del proyecto.
- **.gitignore**: indica qué archivos no deben subirse al repositorio.

## Requisitos

- Python 3.12 o superior.
- Las librerías indicadas en `requirements.txt`.
